COMUNICACION HTTP

La comunicación entre la aplicación y el cubo XEMOS se lleva a cabo mediante HTTP, utilizando un servidor JSON implementado con Node.js.

Una vez el cubo manda los datos, la aplicación envía solicitudes HTTP al servidor, solicitando los datos almacenados.

El servidor procesa estas solicitudes y responde con los datos solicitados en formato JSON, que luego son interpretados y visualizados por la aplicación para brindar al usuario una experiencia de monitoreo en tiempo real.
