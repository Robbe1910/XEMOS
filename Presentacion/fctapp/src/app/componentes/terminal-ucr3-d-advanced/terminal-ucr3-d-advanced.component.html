XEMOS - ESP32 T7 S3

El prototipo anterior se trata de un modelo de pruebas, ya que para que sea móvil requiere de ser inalámbrico, mientras que nuestro modelo de pruebas aún es alámbrico.

La idea principal sería crear un modelo totalmente móvil, el cuál funcionará con un modelo de ESP32 con una arquitectura diferente al que actual usamos, el cuál nos ofrece la posibilidad de alimentarlo mediante baterías .

Este se denomina LYLIGO T7 S3.
