CONEXION & POWER ON

Su arranque es tán sencillo como la conexión a una red Wifi preestablecida en sus ajustes internos.

Aunque parezca un sistema poco dinámico, para el siguiente prototipo sería indicado y acertado generar una web de acceso de configuración para el dispositivo, para poder acceder a sus valores de configuraciones y setear una SSID y PASS concreta.

Con esto conseguimos que el dispositivo tenga acceso a la conexión Wifi y pueda manadar los datos correctamente.
