APP MOVIL - FUNCIONAMIENTO

La aplicación XEMOS ofrece una interfaz intuitiva para visualizar los datos en tiempo real de una base de datos JSON creada con Node.js.

Al abrir la aplicación, los usuarios pueden ver un resumen de los datos clave, como temperatura, humedad, calidad del aire y pulsaciones del corazón.

Además, la aplicación presenta gráficos claros y fáciles de entender que muestran la tendencia de los datos a lo largo del tiempo, permitiendo a los usuarios monitorear eficazmente el entorno y tomar medidas según sea necesario.
