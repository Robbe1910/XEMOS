CONEXION & POWER ON

Configurar este dispositivo es muy fácil, solo necesita conectarse a una red WiFi preestablecida en sus ajustes internos.

Aunque puede parecer simple, para futuras versiones, sería útil crear una página web para configurarlo, donde puedas cambiar la red Wi-Fi y la contraseña.

Esto asegura que el dispositivo esté conectado a internet y pueda enviar los datos correctamente.
